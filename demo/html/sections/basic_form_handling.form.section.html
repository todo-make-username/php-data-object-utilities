Description:

These are some examples of various ways of hydrating an object from form values.

Bools are where this gets fun as PHP has smart conversions for it. yes/no, 1/0, 'true'/'false' all cleanly convert to boolean. Setting the strict flag to false on a bool field will allow truthy/falsey values. For everyone's sanity, values that are set but empty are not affected by the strict flag and are simply converted to false.

It is highly recommended to set a default value for any inputs that don't always get sent over, like checkboxes. Otherwise PHP might start yelling at you about accessing uninitialized properties.

An optional ConversionSettings attribute can be set on a property to allow loose conversions. An example can be seen below.

By default, all conversions are strict, meaning the value before must loosely match the value after '123' == 123.
