This is an example of auto conversion is used during hydration. By default, all conversions are strict, meaning the value before must loosely match the value after '123' == 123.

An optional Conversion attribute can be set on a property to allow loose conversions. An example can be seen below.

Bools are where this gets fun as PHP has smart conversions for it. yes/no, 1/0, 'true'/'false' all cleanly convert to boolean. Setting the strict flag to false on a bool field will allow truthy/falsey values. For everyone's sanity, empty values are not affected by the strict flag and are simply converted to false.

Arrays are converted only from strings. They can either be csv or json.
