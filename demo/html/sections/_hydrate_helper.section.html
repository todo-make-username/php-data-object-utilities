Description:

Here are some examples of how the ObjectHydrator is used to hydrate an object's public properties using $_POST.

Usage:

$HydratedObj = (new ObjectHydrator($Obj))
			->hydrate($_POST)
			->getObject();
		
$HydratedObj = (new ObjectHydrator())
			->setObject($Obj)
			->hydrate($_POST)
			->getObject();
