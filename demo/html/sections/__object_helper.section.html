Description:

This is a helper class that you can use as a wrapper for all the functionality in this library.

For a practical example: The examples in this demo use this helper submit.php.

Usage Example:

$ObjectHelper = new ObjectHelper($Obj);
$is_valid     = $ObjectHelper
        ->hydrate($_POST)
	->tailor()
	->isValid();
$NewObj       = $ObjectHelper->getObject();
$messages     = $ObjectHelper->getValidatorMessages();
