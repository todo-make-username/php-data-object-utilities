Description:

Here are some examples of how the ObjectValidator is used to validate an object's public properties using various attributes.

validate() and isValid() do the same thing.

Validation attributes are processed in order from top to bottom and each one should pass in a failure message into the constuctor.

Quick tip: One of the more useful combinations is using Trim followed up by NotEmpty.

Usage:

$is_valid = (new ObjectValidator($Obj))
			->isValid();
		
$is_valid = (new ObjectValidator())
			->setObject($Obj)
			->validate();
		
$ObjectValidator = new ObjectValidator($Obj);
$is_valid = $ObjectValidator->isValid();

$message = ($is_valid) ? 'Success' : $ObjectValidator->getMessage();
